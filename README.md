# AI-Powered SEO Research Project (2024-2026)

## 🎯 Project Objective
This repository serves as a technical research hub for identifying and documenting cutting-edge **Generative Engine Optimization (GEO)** and **Answer Engine Optimization (AEO)** strategies. The goal is to move beyond generic AI content and capture the specific "practitioner-level" tactics used by agency founders and growth heads to maintain search visibility in an AI-first era.

## 🛠️ Technical Stack
- **Languages:** Python 3.10+
- **APIs:** [Supadata API](https://supadata.ai/) (YouTube Transcript Retrieval)
- **Data Processing:** Custom Regex Cleaning Pipeline (Filler word removal & readability formatting)
- **Architecture:** Local-first Markdown repository structured by Author and Content Pillar.

## 🔍 Selection Criteria: The 'High-Signal' Standard
Unlike general influencers, the 15 experts tracked in this repository were selected based on a **Signal-to-Noise** evaluation (Scored 1-10):
1. **Raw Data & Tactics:** They share specific prompts, vector embedding strategies, or A/B test results.
2. **Active Agency/Founder Status:** They are currently managing real-world client portfolios or building SEO software.
3. **Future-Proofing:** Their insights focus on 2024-2026 trends, specifically how LLMs (Gemini, Perplexity, SearchGPT) ingest and cite data.

## 🚀 Getting Started
1. **Setup Environment:** `python -m venv venv` and `.\venv\Scripts\activate`.
2. **Install Tools:** `pip install -r requirements.txt`.
3. **Collect Data:** Run `python scripts/fetch_research.py` to fetch new transcripts or manually add LinkedIn insights.
